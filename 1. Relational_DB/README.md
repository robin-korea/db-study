# 1. 관계형 데이터베이스 (Relational DB) 학습

이 저장소는 관계형 데이터베이스(RDBMS)의 핵심 개념을 학습하고, 다양한 RDBMS(Oracle, MySQL 등)를 직접 설치하고 실습한 내용을 정리한 공간입니다.
