# 📂 미니 프로젝트 (Mini Projects)

이 폴더는 다양한 관계형 데이터베이스(RDBMS)를 사용하여 학습한 개념들을 종합적으로 활용한 결과물들을 모아둔 공간입니다. 
데이터베이스 시스템을 직접 **설계 및 구현**하는 프로젝트와, 복잡한 비즈니스 요구사항을 해결하는 **실전 쿼리 트레이닝**으로 구성되어 있습니다.

---

## 🚀 프로젝트 목록

### 1. Oracle 기반 도서 대출 관리 시스템 (`Oracle_Library_Project`)
- **개요**: Oracle DB를 사용하여 도서 정보, 회원 정보, 대출/반납 기록을 관리하는 시스템입니다.
- **주요 학습 내용**: 테이블 설계(DDL), 제약조건, 시퀀스, PL/SQL 프로시저(대출/반납 로직), 분석 함수(랭킹), 뷰(View) 등 Oracle의 핵심 기능을 종합적으로 활용했습니다.

### 2. PostgreSQL 기반 상품 리뷰 시스템 (`PostgreSQL_Review_Project`)
- **개요**: PostgreSQL DB를 사용하여 상품 정보와 사용자 리뷰를 관리하는 시스템입니다.
- **주요 학습 내용**: PostgreSQL의 고유 기능인 `SERIAL`, `ARRAY`, `JSONB` 데이터 타입을 적극적으로 활용하여 유연한 데이터 구조를 설계했습니다. PL/pgSQL 프로시저를 통해 리뷰 등록 시 상품의 평균 평점이 자동으로 업데이트되는 기능을 구현했습니다.

### 3. 실전 SQL 쿼리 트레이닝 (`SQL_Practice_Set`)
- **개요**: 실제 서비스와 유사한 **음원 스트리밍(Spotify)** 및 **대용량 도서(Books)** 데이터를 기반으로, 총 36개의 복잡한 비즈니스 시나리오를 SQL로 해결한 실습 과제입니다.
- **주요 학습 내용**: 
    * **Spotify:** 다대다(N:M) 관계 테이블을 연결하는 **Advanced JOIN** 기법 숙달.
    * **Books:** **CTE(Common Table Expression)**와 **Window Functions**(RANK, PARTITION BY)를 활용한 심화 데이터 분석 및 통계 추출.

---

## 📂 폴더 구조 안내
각 프로젝트 폴더는 다음과 같은 파일들로 구성되어 있습니다.

* **DDL/DML**: 데이터베이스 스키마 정의 및 초기 데이터 셋업.
* **Procedures**: 비즈니스 로직을 처리하는 저장 프로시저.
* **Analysis/Questions**: 데이터 분석 쿼리 및 문제 해결 코드.
